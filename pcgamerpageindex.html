les composants nécessaires pour monter son PC Gaming

Guide étape par étape

Carte mère

La base de ton ordinateur. Elle relie tous les composants entre eux.

Processeur (CPU)

Le cerveau de la machine. Il exécute toutes les instructions.

Mémoire RAM

Permet de stocker temporairement les données pour un accès rapide.

Carte graphique

Indispensable pour les jeux et le rendu visuel haute performance.

Alimentation

Fournit l'énergie à tous les composants du PC.

Boîtier

Protège les composants et assure une bonne ventilation.
